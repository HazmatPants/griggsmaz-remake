# Griggsmaz Remake

![img](https://img.itch.zone/aW1nLzIyODYwOTQxLnBuZw==/original/RIpDmN.png)

A game created with [Godot Engine](https://github.com/godotengine/godot).

Builds are currently not available. However, you can build it yourself, clone this repo and import it into Godot, then go to `Project > Export`.
